# Railway API 키 업데이트 가이드

## 총 6개 API 키 설정

Railway 대시보드에서 `CLAUDE_API_KEYS` 환경 변수를 다음 값으로 업데이트하세요:

### 복사할 값 (전체)

**⚠️ 중요: 실제 API 키는 Railway 대시보드에서 직접 입력하세요.**

```json
["키1","키2","키3","키4","키5","키6"]
```

**실제 키 형식:**
- 키는 `sk-ant-api03-`로 시작하는 긴 문자열입니다
- 총 6개의 키를 쉼표로 구분하여 배열로 입력하세요

## Railway 업데이트 방법

1. **Railway 대시보드 접속**
   - https://railway.app 접속
   - 프로젝트 선택 (Project ID: `syfebybkepvvlgdpkzfu`)

2. **Variables 탭으로 이동**
   - 프로젝트 → **Variables** 탭 클릭

3. **CLAUDE_API_KEYS 변수 찾기**
   - `CLAUDE_API_KEYS` 변수를 찾아서 클릭

4. **값 업데이트**
   - 위의 전체 JSON 배열 값을 복사하여 붙여넣기
   - **Save** 버튼 클릭

5. **배포 확인**
   - Railway가 자동으로 재배포를 시작합니다
   - **Deployments** 탭에서 배포 진행 상황 확인

## 키 목록 (참고용)

**⚠️ 실제 API 키는 보안상 이 문서에 포함하지 않습니다.**
- Railway 대시보드에서 직접 입력하거나
- 로컬에서만 관리하는 별도 파일에 보관하세요

## 주의사항

⚠️ **중요**: 
- JSON 배열 형식이어야 합니다 (큰따옴표 포함)
- 각 키는 큰따옴표로 감싸야 합니다
- 쉼표로 구분하되, 마지막 키 뒤에는 쉼표가 없어야 합니다
- 공백 없이 한 줄로 입력하거나, Railway가 자동으로 포맷팅해도 됩니다

## 확인 방법

업데이트 후 배포가 완료되면:
1. 배포된 사이트에서 원고 생성 테스트
2. 여러 작업을 동시에 생성하여 라운드로빈이 정상 작동하는지 확인
3. Railway 로그에서 API 호출이 정상적으로 이루어지는지 확인
